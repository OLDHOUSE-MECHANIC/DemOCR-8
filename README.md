# ScreenReader

**Crop any part of your screen → OCR → Spoken aloud. 100% offline. Free forever.**

Works on Debian Trixie (and any modern Debian/Ubuntu). Optimised for low-spec hardware.

---

## Features

- **Drag a crop box** over any part of your screen (like a screenshot tool)
- **Reads any text** — normal text, text inside images, logos, memes, screenshots
- **Image preprocessing** — contrast boost + sharpening + adaptive threshold so embedded/stylised text is detected
- **Piper TTS** — natural-sounding neural voice, completely offline
- **Falls back to espeak-ng** automatically if Piper isn't installed
- **Global hotkey**: `Ctrl + Shift + S` — works in any app
- **No API. No internet. No cost.**

---

## Install (once)

```bash
cd screenreader/
bash setup.sh
```

This will:
1. Install `tesseract-ocr`, `espeak-ng`, Python packages via `apt`
2. Download **Piper TTS** binary + `en_US-amy-medium` voice (~70 MB one-time)
3. Create `run.sh` launcher
4. Add a `.desktop` entry (shows in your app menu)
5. Add autostart so it launches on login

---

## Run

```bash
bash run.sh
```

Or search **ScreenReader** in your application menu.

---

## Usage

1. Press **`Ctrl + Shift + S`** anywhere
2. Your screen dims — drag a rectangle over the text you want read
3. Release mouse — text is extracted and spoken aloud
4. Press **ESC** to cancel the selection

---

## Troubleshooting

| Issue | Fix |
|---|---|
| No voice output | `sudo apt install espeak-ng alsa-utils` |
| OCR misses text | Try selecting a larger region or zoom in first |
| Hotkey not working | Make sure `screenreader.py` is running (check taskbar) |
| Piper voice not found | Re-run `bash setup.sh` — it will re-download |
| `aplay: command not found` | `sudo apt install alsa-utils` |

## Changing the voice

After setup, Piper voices live in `~/.local/share/piper-voices/`.
Download any voice from https://github.com/rhasspy/piper#voices and place the `.onnx` + `.onnx.json` there.
The program auto-detects the first `.onnx` it finds.

## Changing the hotkey

Open `screenreader.py` and edit line:

```python
"hotkey": "<ctrl>+<shift>+s",
```

(Note: the hotkey string in CONFIG is for reference; actual detection is in `on_press()`)
